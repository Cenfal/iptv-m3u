# iptv-m3u

A curated M3U playlist for IPTV players.

## Playlist

**File:** `playlist.m3u`

### Channels

| Channel | Group | URL |
|---------|-------|-----|
| Al Mayadeen English | News | https://english.almayadeen.net/live |
| Al Manar | News | https://www.almanar.com.lb/live/ |

## Usage

Load `playlist.m3u` in any IPTV player that supports the M3U format (e.g. VLC, Kodi, TiviMate, IPTV Smarters).

> **Note:** The stream URLs point to live web pages. If your IPTV player requires a direct HLS/RTMP stream URL, you may need to extract the underlying `.m3u8` endpoint from each site.
